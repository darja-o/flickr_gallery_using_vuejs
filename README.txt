The following solution uses Flickr API, Javacript and Vue.js framework to search flickr photos by keyword.

Vue.js is a JavaScript library for building web interfaces. In this task it was used due to its lightweitness and ease-of-use. Another option was JQuery but it was decided that Vue.js is preferable as it can do the job and do not aggravates performance.

To run the project open index.html file in any modern web browser.
